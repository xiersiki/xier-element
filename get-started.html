最新 Vue3 + TS 高仿 ElementPlus 打造自己的组件库 ​

安装 ​

bash
npm i @xier-element --save

开始使用 ​

全局使用

js
// 引入所有组件
import XrElement from 'xier-element'

import App from './App.vue'
// 全局使用
createApp(App).use(XrElement).mount('#app')
vue
<template>
  <xr-button>我是 Button</xr-button>
</template>

单个导入

XrElement 提供了基于 ES Module 的开箱即用的 Tree Shaking 功能。

vue
<template>
  <xr-button>我是 Button</xr-button>
</template>
<script>
  import { ErButton } from ' xr-element'
  export default {
    components: { xrButton },
  }
</script>

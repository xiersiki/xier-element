Message 全局消息 ​

用于在页面中以浮层方式展示短时消息提示，支持多种类型、自动关闭、关闭按钮和自定义时长。

Message 为全局 API 风格（函数调用），无需在模板中注册组件实例。

基础用法 ​

通过调用 XrMessage 显示消息：

基本消息

通过调用 XrMessage 显示消息。

<template>
    <div class="message-demo">
        <h3>基本消息</h3>
        <p>通过调用 <code>XrMessage</code> 显示消息。</p>

        <div class="message-buttons">
            <xr-button @click="showMessage">显示消息</xr-button>
        </div>
    </div>
</template>

<script setup lang="ts">
import { XrButton, XrMessage } from 'xier-element'

const showMessage = () => {
    XrMessage('这是一条消息')
}
</script>

<style scoped>
.message-demo {
    padding: 20px;
}

.message-buttons {
    margin-top: 10px;
}
</style>

不同类型 ​

Message 支持多种类型：success、warning、error、info。

不同类型

Message 支持多种类型：success、warning、error、info。

<template>
    <div class="message-demo">
        <h3>不同类型</h3>
        <p>Message 支持多种类型：success、warning、error、info。</p>

        <div class="message-buttons">
            <xr-button @click="showSuccess" type="success">成功消息</xr-button>
            <xr-button @click="showWarning" type="warning" style="margin-left: 10px;">警告消息</xr-button>
            <xr-button @click="showError" type="danger" style="margin-left: 10px;">错误消息</xr-button>
            <xr-button @click="showInfo" style="margin-left: 10px;">信息消息</xr-button>
        </div>
    </div>
</template>

<script setup lang="ts">
import { XrButton, XrMessage } from 'xier-element'

const showSuccess = () => {
    XrMessage.success('成功消息')
}

const showWarning = () => {
    XrMessage.warning('警告消息')
}

const showError = () => {
    XrMessage.error('错误消息')
}

const showInfo = () => {
    XrMessage('信息消息')
}
</script>

<style scoped>
.message-demo {
    padding: 20px;
}

.message-buttons {
    margin-top: 10px;
}
</style>

自定义配置 ​

通过配置对象自定义消息的显示时长、关闭按钮等。

自定义配置

通过配置对象自定义消息的显示时长、关闭按钮等。

<template>
    <div class="message-demo">
        <h3>自定义配置</h3>
        <p>通过配置对象自定义消息的显示时长、关闭按钮等。</p>

        <div class="message-buttons">
            <xr-button @click="showCustom">自定义消息</xr-button>
            <xr-button @click="showLong" style="margin-left: 10px;">长时显示</xr-button>
            <xr-button @click="showClosable" style="margin-left: 10px;">可关闭</xr-button>
        </div>
    </div>
</template>

<script setup lang="ts">
import { XrButton, XrMessage } from 'xier-element'

const showCustom = () => {
    XrMessage({
        message: '自定义消息',
        type: 'info',
        duration: 5000
    })
}

const showLong = () => {
    XrMessage({
        message: '这条消息会显示 5 秒',
        type: 'success',
        duration: 5000
    })
}

const showClosable = () => {
    XrMessage({
        message: '点击右上角关闭',
        type: 'warning',
        showClose: true
    })
}
</script>

<style scoped>
.message-demo {
    padding: 20px;
}

.message-buttons {
    margin-top: 10px;
}
</style>

居中显示 ​

设置 center 属性使消息居中显示。

居中显示

设置 center 属性使消息居中显示。

<template>
    <div class="message-demo">
        <h3>居中显示</h3>
        <p>设置 <code>center</code> 属性使消息居中显示。</p>

        <div class="message-buttons">
            <xr-button @click="showCenter">居中消息</xr-button>
        </div>
    </div>
</template>

<script setup lang="ts">
import { XrButton, XrMessage } from 'xier-element'

const showCenter = () => {
    XrMessage({
        message: '这条消息居中显示',
        type: 'info',
        center: true,
        duration: 3000
    })
}
</script>

<style scoped>
.message-demo {
    padding: 20px;
}

.message-buttons {
    margin-top: 10px;
}
</style>

配置与 API ​

MessageOptions ​

名称说明类型默认
message消息内容，可为字符串、VNode 或渲染函数MessageContent-
type消息类型'info' | 'success' | 'warning' | 'error''info'
duration自动关闭时长（毫秒），为 0 则不自动关闭number3000
showClose是否显示关闭按钮booleanfalse
center是否居中显示（水平居中）booleanfalse
zIndex覆盖的 z-index 值number-

MessageParams ​

消息参数可以直接传入字符串、VNode、或 MessageOptions 对象，库内部会进行适配。

MessageHandler ​

调用返回的 handler 对象允许在代码中手动关闭该条消息：

方法说明
close()关闭当前消息

说明 ​

  • Message 是全局 API，通常通过导入后直接调用；无需在组件模板中声明。
  • 若需同时控制多条消息，可在业务层保存返回的 handler 对象进行关闭。

TIP

如果需要更复杂的消息样式或交互，可通过自定义 VNode 或渲染函数传入 message 字段，实现任意展示内容。
